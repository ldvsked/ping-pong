Игра Pong для терминала

## О проекте

Реализация классической игры Pong для терминала. Проект представляет собой пошаговую версию игры с ASCII-графикой, написанную на языке C с использованием только стандартной библиотеки.

## Игровой процесс

- **Игровое поле**: прямоугольник размером 80×25 символов
- **Ракетки**: размер 3 символа, управление:
  - Левая ракетка: клавиши A (вверх) и Z (вниз)
  - Правая ракетка: клавиши K (вверх) и M (вниз)
- **Мяч**: размер 1 символ
- **Пропуск хода**: пробел
- **Счет**: отображается на экране в реальном времени
- **Победа**: игра продолжается до 21 очка, после чего отображается поздравление победителя

## Режим работы

Программа работает в пошаговом режиме:
1. Ожидание ввода от игрока (движение ракетки или пропуск хода)
2. Отрисовка обновленного состояния поля
3. Повтор до завершения игры

## Технические требования и особенности реализации

- **Язык**: C
- **Графика**: только ASCII-символы (|, _, ‾, :, ©)
- **Структура**: функциональный подход с отдельными функциями для каждого действия
- **Обработка ввода**: посимвольное считывание с очисткой буфера
- **Управление**: 
  - A/Z — движение левой ракетки
  - K/M — движение правой ракетки
- **Отрисовка поля**: двойной цикл for для построчного вывода
- **Логика коллизий**: 
  - Отскок от ракеток при совпадении координат
  - Отскок от верхней и нижней границ
- **Подсчет очков**: при выходе мяча за левую/правую границу

## Используемые конструкции

- Условные операторы (if/else)
- Циклы (while, for)
- Функции с возвращаемыми значениями
- Форматированный ввод/вывод (scanf/printf)
